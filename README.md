## Installation

**Required:** Node.js

Clone Repository

```bash
  git clone https://github.com/mrawsky/CryptoCracker.git
  cd CryptoCracker
```

Install Node Packages

```bash
  npm i
```

Run CryptoCracker

```bash
  npm start
```
